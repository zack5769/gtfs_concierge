# 背景画像の設定

このディレクトリに `background.jpg` という名前で画像ファイルを配置してください。

## 推奨仕様
- ファイル名: `background.jpg`（または `background.png`）
- 推奨サイズ: 1920x1080px 以上
- フォーマット: JPG, PNG

## 使用方法
1. お好みの画像を用意
2. このディレクトリ（`public/`）に `background.jpg` として配置
3. アプリを再起動（開発サーバーを再起動）

画像が配置されると、ホーム画面の背景として表示されます。
画像の上には半透明の白いオーバーレイ（85%の不透明度）がかかり、テキストが読みやすくなります。

## 画像フォーマットの変更
PNG形式を使用する場合は、`src/pages/Home.css` の以下の行を変更してください：

```css
background-image: url('/background.png');
```
